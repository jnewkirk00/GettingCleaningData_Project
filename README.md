# GettingCleaningData_Project
Repository for class project

## Script
### Name: run_analysis.r

### Purpose: This script is designed to clean and prepare data associated with wearable device measurements - specifically the Samsung data described [here](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones).

### Output: The output of the script is a summary text file that groups wearable mean and std measurement data by Subject (person) and Activity (walking, standing, etc.).
